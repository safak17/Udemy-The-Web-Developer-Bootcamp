```css
h1{
  color: red;
  color: #FF0000; 
  color: rgb(255, 0, 0);
  color: rgba(255, 0, 0, .6);
  //	RGBA is just like RGB but with an alpha(transparency) channel. Ranges from 0.0 - 1.0
  
  border-width: 2px;
  border-style: solid;
  border-color: purple;
  
  border: 2px solid purple;
}
```
